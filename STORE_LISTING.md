## Deutsch

### Kurzbeschreibung (max 100 Zeichen)
Leichtgewichtiger SQLite-Browser — Datenbanken oeffnen, durchsuchen und abfragen ohne Installation.

### Beschreibung (max 10.000 Zeichen)
SQLite Viewer Pro ist ein schneller, portabler SQLite-Datenbank-Browser. Er oeffnet jede .db-, .sqlite- oder .sqlite3-Datei sofort und ermoeglicht das Durchsuchen, Abfragen und Exportieren von Daten — ohne SQL-Kenntnisse und ohne externe Abhaengigkeiten.

**Was SQLite Viewer Pro kann:**

- Tabellen-Browser: Listet automatisch alle Tabellen auf und zeigt Daten in einem sortierbaren Raster an
- Schema-Ansicht: CREATE TABLE Statements mit Syntax-Highlighting anzeigen
- SQL-Editor: Eigene SQL-Abfragen schreiben und ausfuehren mit Syntax-Highlighting und Ergebnisansicht
- Volltextsuche: Echtzeit-Suche ueber alle Spalten — einfach lostippen
- CSV-Export: Tabellen oder Abfrageergebnisse als CSV-Datei exportieren
- Sortierung: Spaltenkoepfe anklicken fuer auf- oder absteigende Sortierung
- Tastenkuerzel: Ctrl+O (Oeffnen), Ctrl+F (Suche), Ctrl+E (Export), F5 (Aktualisieren), F9 (SQL ausfuehren)

**Warum SQLite Viewer Pro statt DB Browser oder DBeaver?**

- Sofortiger Start: Keine lange Ladezeit, kein Java, kein Framework
- Portabel: Eine einzige Python-Datei, laeuft ueberall wo Python installiert ist
- Leichtgewichtig: ~750 Zeilen Code, minimaler Speicherverbrauch
- Keine Installation noetig: Starten und sofort nutzen
- Volltextsuche ueber alle Spalten in Echtzeit

**Fuer wen ist SQLite Viewer Pro?**

Entwickler, Datenanalysten und Power-User, die schnell einen Blick in SQLite-Datenbanken werfen moechten. Ideal als Ersatz fuer schwergewichtige Datenbank-Tools bei einfachen Aufgaben.

## English

### Short Description (max 100 chars)
Lightweight SQLite browser — open, search and query databases instantly, no installation needed.

### Description (max 10,000 chars)
SQLite Viewer Pro is a fast, portable SQLite database browser. It opens any .db, .sqlite, or .sqlite3 file instantly and lets you browse, query, and export data — no SQL knowledge required, no external dependencies.

**What SQLite Viewer Pro does:**

- Table Browser: Automatically lists all tables and displays data in a sortable grid
- Schema View: Inspect CREATE TABLE statements with syntax highlighting
- SQL Editor: Write and execute custom SQL queries with syntax highlighting and result view
- Full-Text Search: Real-time search across all columns — just start typing
- CSV Export: Export any table or query result to a CSV file
- Sorting: Click column headers to sort ascending or descending
- Keyboard Shortcuts: Ctrl+O (open), Ctrl+F (search), Ctrl+E (export), F5 (refresh), F9 (execute SQL)

**Why SQLite Viewer Pro over DB Browser or DBeaver?**

- Instant startup: No long loading times, no Java, no framework overhead
- Portable: A single Python file, runs anywhere Python is installed
- Lightweight: ~750 lines of code, minimal memory footprint
- No installation required: Launch and use immediately
- Full-text search across all columns in real time

**Who is SQLite Viewer Pro for?**

Developers, data analysts, and power users who need a quick look into SQLite databases. Ideal as a replacement for heavyweight database tools for simple tasks.
